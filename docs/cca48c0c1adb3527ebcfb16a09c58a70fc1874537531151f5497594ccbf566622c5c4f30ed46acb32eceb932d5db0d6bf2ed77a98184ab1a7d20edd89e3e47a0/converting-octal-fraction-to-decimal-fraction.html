22 Converting octal fraction to decimal fraction

22.1 CONVERT OCTAL TO BINARY

22.1.1 Algorithm

  1. Start

  2. For each digit of the Octal number, write out the binary digits in groups of three

  3. Stop

(540)8

(101100000)2

352.5638

352 = 011101010

0.563 = 101 110 011

352.563 = 011101010.101110011

205 octal to binary

010000101

22.2 BINARY to OCTAL

  1. Start

  2. Separate and group binary number into sets of three, starting with the least significant (rightmost digit)

  3. Pad the Most Significant Digit (leftmost) with zeros, if necessary, to make a group of three

  4. Write out the corresponding Octal digits for each set of three

  5. Stop

Convert to octal 1110111111

001 110 111 111

1677 octal

Convert to octal Binary: 011 100 101

Octal: 3 4 5

22.3 BINARY TO HEXADECIMAL

  1. Start

  2. Separate and group binary number into sets of four, starting with the least significant (rightmost digit)

  3. Pad the Most Significant Digit (leftmost) with zeros, if necessary, to make a group of four

  4. Write out the corresponding Octal digits for each set of four

  5. Stop

Convert to Hexadecimal 1110111111

0011 1011 1111 = 3BF

Convert binary 1110 0101 to hexa

1110 = E and 0101 = 5

Hence Binary: 1110 0101 = Hexa : E5

22.4 HEXADECIMAL to BINARY

Algorithm

  1. Start

  2. For each digit of the Octal number, write out the binary digits in groups of four

  3. Stop

Examples

3BF = 0011 1011 1111

Convert hexadecimal BAD into binary

1011 1010 1101

Convert BEEF to binary 1011 1110 1110 1111

22.5 OCTAL to HEXADECIMAL

Algorithm

  1. Start

  2. Convert Octal to binary

  3. Drop any zeros at the leftmost end

  4. Convert binary to Hexa

  5. Stop

Convert Octal 345 to hexadecimal

Octal: 345 = 011100101 binary

After dropping zero at leftmost end Binary: 11100101

Separate into sets of 4 digits : 1110 0101 = E5
