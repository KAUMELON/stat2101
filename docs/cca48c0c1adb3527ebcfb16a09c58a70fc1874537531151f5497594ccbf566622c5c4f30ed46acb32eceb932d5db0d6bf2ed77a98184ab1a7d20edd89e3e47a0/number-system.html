8.2 CONVERSION OF DECIMAL TO BINARY

8.2.1 Algorithm for converting decimal to binary

  1. Start

  2. Divide the given integer by 2 and write down the quotient & remainder

  3. Divide the quotient by 2 and record the second quotient and remainder. Repeat the same steps until the quotient = 0

  4. Write out the remainders in reverse order to get binary digit of the integer

  5. Stop

8.2.2 Examples

  1. Convert decimal 24 to binary
2 24 0
2 12 0
2 6 0
2 3 1
2 1 1
0

2410 = 110002

  1. Convert 1510 to binary
2 15 1
2 7 1
2 3 1
2 1 1
0

Binary digit is 1111

Hence 1510 = 11112

8.3 CONVERT BINARY TO DECIMAL

8.3.1 Algorithm

  1. Start

  2. Start by multiplying the rightmost digit with 20

  3. Right to left, multiply each digit of the binary number with increased power of 2, until no more digits are left

  4. Add the products that are obtained

  5. Stop

8.3.2 Examples

Convert binary to decimal

  1. 10012 to decimal

1 x 23 + 0 x 22 + 0 x 21 + 1 x 20 ; 1 x 8 + 0 x 4 + 0 x 2 + 1 x 1 ; 8+0+0+1 = 9; 10012 = 910

8.4 CONVERTING BINARY FRACTION TO DECIMAL FRACTION

8.4.1 Algorithm

  1. Start

  2. Starting from left most digit to right, multiply each digit by powers of (1/2), starting with the power 1

  3. Add the results obtained to get the decimal fraction

  4. Stop

8.4.2 Examples

Convert binary fraction to decimal fraction

  1. 0.1012

1x (1/2)1 + 0 x (1/2)2 + 1 x (1/2)3

1 x 0.5 + 0 x 0.25 + 1 x 0.125 = 0.5 + 0 + 0.125 = 0.625

0.1012 = 0.62510

  1. 0.0112

0 x (1/2) + 1 x (1/2)2 + 1 x (1/2)3 = 0 + 0.25 +0.125 = 0.37510

8.5 CONVERTING DECIMAL FRACTION TO BINARY FRACTION

8.5.1 Algorithm

  1. Start

  2. Multiply the decimal fraction by 2. From the product, separate the integral part and note down

  3. To the decimal part of the product with multiply once again with 2. Note down the integral part. And multiply with 2 the fraction/decimal part.

  4. Repeat the process until we obtain the 2 or three or more digits after decimal point as required

  5. The integral part is written down from top to bottom to obtain the binary fraction

  6. Stop

8.5.2 Example

Convert 0.4710 to binary fraction

0.47 x 2 = 0.94 integral part = 0

0.94 x 2 = 1.88 integral part = 1

0.88 x 2 = 1.76 integral part = 1

0.76 x 2 = 1.52 integral part = 1

0.52 x 2 = 1.04 integral part = 1

0.04 x 2 = 0.08 integral part = 0

0.4710 = 0.0111102

Convert 111001.1012 to decimal digit

1 x 25 + 1 x 24 x 1 x 23 + 0 x 22 + 0 x 21 +1 x 20

=57.625

Convert 33225.11210 to binary

Answer: 1000000111001001. 000111001010110

8.8 HEXADECIMAL TO DECIMAL

8.8.1 Algorithm

  1. Start

  2. Start by multiplying the rightmost digit with 160

  3. Right to left, multiply each digit of the binary number with increased power of 16, until no more digits are left

  4. Add the products that are obtained

  5. Stop

A2DE hexadecimal: = ((A) * 163) + (2 * 162) + ((D) * 161) + ((E) * 160)
= (10 * 163) + (2 * 162) + (13 * 161) + (14 * 160)
= (10 * 4096) + (2 * 256) + (13 * 16) + (14 * 1)
= 40960 + 512 + 208 + 14
= 41694 decimal

8.9.1 CONVERT OCTAL TO BINARY

8.9.1.1 Algorithm

  1. Start

  2. For each digit of the Octal number, write out the binary digits in groups of three

  3. Stop

(540)8

(101100000)2

352.5638

352 = 011101010

0.563 = 101 110 011

352.563 = 011101010.101110011

010000101

8.9.2 BINARY to OCTAL

  1. Start

  2. Separate and group binary number into sets of three, starting with the least significant (rightmost digit)

  3. Pad the Most Significant Digit (leftmost) with zeros, if necessary, to make a group of three

  4. Write out the corresponding Octal digits for each set of three

  5. Stop

Convert to octal 1110111111

001 110 111 111

1677 octal

Octal: 3 4 5

8.9.3 BINARY TO HEXADECIMAL

  1. Start

  2. Separate and group binary number into sets of four, starting with the least significant (rightmost digit)

  3. Pad the Most Significant Digit (leftmost) with zeros, if necessary, to make a group of four

  4. Write out the corresponding Octal digits for each set of four

  5. Stop

Convert to Hexadecimal 1110111111

0011 1011 1111 = 3BF

Convert binary 1110 0101 to hexa

1110 = E and 0101 = 5

Hence Binary: 1110 0101 = Hexa : E5

8.9.4 HEXADECIMAL to BINARY

8.9.4.1 Algorithm

  1. Start

  2. For each digit of the Octal number, write out the binary digits in groups of four

  3. Stop

8.9.4.2 Examples

3BF = 0011 1011 1111

Convert hexadecimal BAD into binary

1011 1010 1101

Convert BEEF to binary 1011 1110 1110 1111

8.9.5 OCTAL to HEXADECIMAL

8.9.5.1 Algorithm

  1. Start

  2. Convert Octal to binary

  3. Drop any zeros at the leftmost end

  4. Convert binary to Hexa

  5. Stop

Convert Octal 345 to hexadecimal

Octal: 345 = 011100101 binary

After dropping zero at leftmost end Binary: 11100101

Separate into sets of 4 digits : 1110 0101 = E5
